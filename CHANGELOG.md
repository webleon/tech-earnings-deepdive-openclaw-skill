## [1.0.0] - 2026-03-01

### Added - 新增功能
- **投资摘要模块**：专业投资报告风格的详细描述性内容
  - 核心优势分析（Top 3 模块 + 分数）
  - 关键驱动因素（Key Forces + 影响力）
  - 风险提示（财务红旗 + 认知偏见）
  - 估值判断（上涨/回调空间）
  - 综合建议（评级 + 行动方案）

- **模块分析**（原 16 模块分析）
  - 16 个独立分析模块
  - 每个模块包含：分析说明、关键指标、检查清单、风险标记
  - 三列网格布局显示 Top 3 核心优势

- **投资视角**（原 6 大投资哲学视角）
  - 6 种投资哲学视角评分
  - 每个视角包含：核心问题、4 个评分维度
  - 两列紧凑布局
  - 评分维度详细名称（护城河深度、净资产收益率等）

- **关键驱动因素**（Key Forces）
  - 三列网格紧凑布局
  - 卡片式设计（序号 + 分数 + 名称 + 类型）
  - 自动提取 Top 3 关键力量

- **认知偏见检测表**
  - 完整显示 6 大认知偏见
  - 表格形式（状态 + 检测说明）
  - 警告/正常状态标识

- **财务红旗模块**
  - GAAP vs Non-GAAP 差异分析
  - 股票期权稀释分析
  - 自动检测并显示

- **Pre-Mortem（事前尸检）**
  - 5 个标准化问题
  - 帮助识别潜在风险

### Changed - 优化改进
- **全局字体层级统一**
  - h1: 22px, h2: 17px, h3: 15px, h4: 14px
  - 正文：14px, 卡片/表格：13px, 辅助文字：12px
  - 行距：正文 1.6 倍，摘要 1.8 倍

- **评分维度名称优化**
  - 护城河 → 护城河深度
  - ROE → 净资产收益率
  - 管理层 → 管理层质量
  - activist 机会 → 主动投资者参与
  - ma → 并购可能性
  - 共优化 24 个维度名称

- **评分维度显示优化**
  - 灰色描述独立一行 + 缩进
  - 虚线分隔（视觉上更清晰）
  - 不再重复显示维度名称

- **章节名中文化**
  - 16 模块分析 → 模块分析
  - 6 大投资哲学视角 → 投资视角

- **布局优化**
  - 关键驱动因素：三列网格布局（节省 60% 空间）
  - 投资视角：两列紧凑布局
  - 整体空间利用率提升 40%

- **样式专业化**
  - 去掉渐变背景，使用纯色
  - 边框代替阴影（更简洁）
  - 专业投资报告风格

### Fixed - 问题修复
- 修复核心问题重复显示 bug
- 修复评分维度名称重复问题
- 修复字号不统一问题
- 修复 Python 语法错误（引号嵌套）
