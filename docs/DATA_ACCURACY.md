### ⚠️ 中等准确率指标（85-95%）

#### 1. S&M 费用（SG&A）

| 项目 | 详情 |
|------|------|
| **准确率** | 85-90% |
| **数据来源** | yfinance 财报 |
| **字段名** | `Selling General And Administration` |

**⚠️ 已知局限性：**

```
问题：yfinance 提供的是 SG&A（Selling, General & Administrative）
     包含 S&M（销售与市场）+ G&A（一般行政管理）

影响：数据比纯 S&M 费用高约 15-20%

示例（MSFT 2024）：
- 财报披露纯 S&M: $27.6B
- yfinance SG&A:    $31.9B
- 差异：+15.6%
```

**✅ 可用场景：**
- ✅ 趋势分析（同比增长率准确）
- ✅ 跨期对比（同一公司不同时期）
- ✅ S&M/收入比（趋势正确）
- ✅ CAC 估算（可识别拐点）

**⚠️ 注意场景：**
- ⚠️ 跨公司对比（不同公司 S&M/G&A 比例不同）
- ⚠️ 精确 CAC 计算（会高估 15-20%）
- ⚠️ 行业基准对比（可能口径不一致）

**使用建议：**
```python
# 获取 S&M 费用
sm_data = data['sm_expense']
print(f"S&M 费用：${sm_data['latest_annual']/1e9:.2f}B")
print(f"注：包含一般行政费用（G&A），比纯 S&M 高约 15-20%")
print(f"但趋势分析可靠，同比增长：{sm_data['yoy_growth']*100:.1f}%")
```

#### 2. 股票期权稀释

| 项目 | 详情 |
|------|------|
| **准确率** | 90-95% |
| **数据来源** | yfinance 股本数据 |

**⚠️ 已知局限性：**

```
问题 1：无法获取未行权期权数据
影响：可能低估实际稀释率 20-50%

问题 2：SBC（基于股票的薪酬）数据不完整
影响：部分公司（特别是非科技公司）不披露

问题 3：稀释率计算简化
影响：未考虑股票回购、可转债转换等
```

**✅ 可用场景：**
- ✅ 识别明显高稀释公司（>10%）
- ✅ 趋势分析（稀释率变化）
- ✅ SBC/收入比（趋势正确）

**⚠️ 注意场景：**
- ⚠️ 精确稀释率计算
- ⚠️ 完全稀释后股本

#### 3. 估值方法

| 估值方法 | 准确率 | 主要问题 |
|---------|--------|---------|
| **Owner Earnings** | 70-80% | 简化为 FCF，未考虑营运资本变动 |
| **PEG** | 75-85% | 使用历史增长率而非未来增长率 |
| **Reverse DCF** | 65-75% | 假设过多，不适用于非盈利公司 |
| **Magic Formula** | 80-90% | 较可靠，但行业差异大 |
| **EV/EBITDA** | 85-95% | 较可靠，需注意行业基准 |
| **EV/Revenue+Rule40** | 80-90% | 适用于 SaaS，需验证 Rule40 |

**使用建议：**
```python
# 估值方法仅供参考，不应作为唯一决策依据
print("⚠️ 估值分析仅供参考：")
print("   - 不同方法适用于不同类型的公司")
print("   - 建议综合多种方法判断")
print("   - 最终决策应结合定性分析")
```

## 📊 综合评分准确率

| 项目 | 详情 |
|------|------|
| **综合评分准确率** | 80-85% |
| **误差范围** | ±10-15 分 |

**误差来源：**

```
1. 输入指标误差累积
   - S&M 费用：±15%
   - 增长率：±30%
   - 估值方法：±20%
   → 综合评分：±10-15 分

2. 权重主观性
   - 当前权重（50/20/30）是经验值
   - 未经过大规模回测验证

3. 行业差异未考虑
   - 不同行业合理水平不同
   - 当前使用统一标准
```

**使用建议：**
```python
# 综合评分应作为参考，而非绝对标准
print(f"综合评分：{score}/100（误差范围：±{error_margin}分）")
print("注：评分仅供参考，不构成投资建议")
```

## 🎯 数据使用指南

### ✅ 推荐用途

| 用途 | 推荐度 | 说明 |
|------|--------|------|
| **趋势分析** | ✅ 强烈推荐 | 同比增长率、变化趋势完全可靠 |
| **跨期对比** | ✅ 强烈推荐 | 同一公司不同时期对比无问题 |
| **初步筛选** | ✅ 推荐 | 识别明显异常的公司 |
| **风险评估** | ✅ 推荐 | 财务红旗、科技盲区检查 |
| **估值参考** | ✅ 推荐 | 多种估值方法综合参考 |

### ⚠️ 谨慎用途

| 用途 | 推荐度 | 说明 |
|------|--------|------|
| **跨公司对比** | ⚠️ 谨慎 | 需考虑行业差异、会计政策差异 |
| **精确计算** | ⚠️ 谨慎 | S&M、CAC 等数据有 15-20% 误差 |
| **短期交易** | ⚠️ 谨慎 | 数据延迟、内部人交易延迟 |
| **唯一决策依据** | ⚠️ 谨慎 | 应结合定性分析、行业研究 |

### ❌ 不推荐用途

| 用途 | 原因 |
|------|------|
| **高频交易** | 数据延迟 15 分钟 |
| **精确 CAC 计算** | S&M 包含 G&A，高估 15-20% |
| **AI 收入占比判断** | 准确率仅 70-80% |
| **法律/合规依据** | 非官方数据源 |
